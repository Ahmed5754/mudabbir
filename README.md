# Mudabbir

Mudabbir is a local-first AI agent with a web dashboard, multi-channel integrations, and tool execution on your own machine.

## Quick Start

```bash
pip install -e .
Mudabbir
```

Then open:

- `http://127.0.0.1:8888`

## Highlights

- Web dashboard (chat, activity, terminal, mission control)
- MCP servers and preset installer
- Multiple channels (Telegram, Discord, Slack, WhatsApp, Matrix, Teams, Google Chat)
- Built-in tools (filesystem, web search, browser, OCR, voice, etc.)
- Local configuration in `~/.Mudabbir/`
